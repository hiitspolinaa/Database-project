 # Database Parking System

This project implements a simple database system for managing parking data.  
It includes tables, constraints and SQL scripts designed for use with Oracle Database.

## Features
- Table creation for parking management  
- Data insertion examples  
- Constraints, primary keys and foreign keys  
- Basic queries for retrieving parking information  

## How to Run
1. Open Oracle SQL Developer (or any Oracle client).  
2. Create a new connection.  
3. Run the script `ParkingSystem.sql`.  
4. All tables and data will be generated automatically.

## Requirements
- Oracle Database  
- SQL Developer
